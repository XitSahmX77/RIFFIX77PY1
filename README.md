# XITSAHMX77 - Legal & Branding Website

> Independent Security Research Organization

## Overview

This is the legal and branding website for XitSahmX77, a security research organization dedicated to ethical hacking, vulnerability research, and responsible disclosure.

## Features

- **Home Page** – Brand introduction, stats, features, and calls to action
- **About Page** – Team information, certifications, research areas
- **Legal Page** – Full legal disclaimer, terms of use, liability disclaimer
- **Policy Page** – Responsible disclosure policy and process

## Tech Stack

- React 19 + Vite
- Tailwind CSS 4
- Framer Motion
- React Router DOM
- Lucide React Icons

## Installation

```bash
npm install
npm run dev
```

## Build

```
npm run build
npm run preview
```

## Deployment

Deploy to Vercel, Netlify, or any static hosting service.
